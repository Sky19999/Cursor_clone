# Cursor Clone

A recreation of the Cursor homepage using HTML and CSS.

## Project Structure

This project consists of two main files:

- **[index.html](./index.html)**: Contains the semantic HTML structure of the homepage, recreating sections like the Hero, Features, Testimonials, and Footer.
- **[style.css](./style.css)**: Contains all the custom styling, CSS variables, and layout definitions (Grid/Flexbox) to match the Cursor design system.
